# Hook 模板库

前 3 秒抓眼球的开头模板。

## 🔢 数字对比型

### 时间对比
- "X 年前我 ____，现在我 ____"
- "用了 X 天 ____，结果 ____"
- "X 小时完成了别人 X 个月的工作"
- "从 X 到 Y，我只用了 Z 天"
- "X 分钟 vs X 小时"

### 数据对比
- "X% 的人不知道 ____"
- "只有 X% 的人做到了 ____"
- "研究表明 X% 的 ____"
- "我测试了 X 个 ____，发现 ____"

### 成本对比
- "花了 $X 学到的东西"
- "X 美元 vs X 万美元"
- "免费做到 ____"

**示例：**
```
"6 个月 vs 6 小时

我花了 6 个月学会编程，
上周用 AI，6 小时完成了同样的项目。"
```

---

## ⚡ 反常识型

### 直接否定
- "不要再 ____ 了"
- "停止做 ____"
- "你不需要 ____"

### 揭露真相
- "____ 是个谎言"
- "____ 正在毁掉你"
- "你被 ____ 骗了"

### 挑战常规
- " Contrarian opinion: ____"
- "Hot take: ____"
- "Unpopular opinion: ____"

**示例：**
```
"不要再学编程了。

2024 年，会提问比会写代码更重要。"
```

---

## 📖 个人故事型

### 困境开场
- "我被 ____ 了"
- "最糟糕的一天教会了我 ____"
- "当我 ____ 时，一切都变了"

### 转变时刻
- "X 年前我不信 ____, 现在我 ____"
- "没人相信我能 ____"
- "他们都以为我疯了"

### 近期事件
- "上周我 ____"
- "昨天发生了一件 ____ 的事"
- "刚刚发现 ____"

**示例：**
```
"我被老板骂了。

'这个需求下周必须上线'

但我不会编程。

今天我用 AI 工具，2 小时搞定了。"
```

---

## ❓ 问题悬念型

### 为什么类
- "为什么 ____?"
- "为什么聪明人 ____?"
- "为什么 X 比 Y 更 ____?"

### 什么类
- "____ 的秘密是什么?"
- "什么是 ____?"
- "____ 的真正含义是?"

### 如何类
- "如何 ____?"
- "怎样在 X 天内 ____?"
- "____ 是怎么做到的?"

### 假设类
- "如果 ____ 怎么办?"
- "如果只能 ____?"
- "当你 ____ 时会发生什么?"

**示例：**
```
"为什么 90% 的创业公司会失败？

不是产品不好，不是市场不对。

真正的原因是..."
```

---

## 🎯 直接利益型

### 方法论
- "X 个方法让你 ____"
- "____ 的终极指南"
- "完整的 ____ 框架"

### 效率承诺
- "用 X 分钟学会 ____"
- "X 步搞定 ____"
- "____ 的捷径"

### 结果导向
- "如何 ____（即使你没有 ____）"
- "没有 ____ 也能 ____"
- "零基础学会 ____"

**示例：**
```
"5 个方法让你的内容 viral：

（即使你只有 100 个粉丝）"
```

---

## 🔥 热点借势型

### 即时跟进
- "[热点事件] 刚刚发生"
- "关于 [热点]，你想知道的 X 件事"
- "[热点] 意味着什么？"

### 独特视角
- "没人告诉你的 [热点] 真相"
- "[热点] 的 X 个隐藏影响"
- "为什么 [热点] 不重要"

**示例：**
```
"GPT-5 发布了。

5 分钟看完重点：

1. ...
2. ...

你想用哪个功能？"
```

---

## 💡 洞察分享型

### 观察发现
- "我注意到 ____"
- "发现了一个 ____ 的模式"
- "X 个公司都有的共同点"

### 思考分享
- "我一直在想 ____"
- "关于 ____ 的思考"
- "____ 教会了我"

**示例：**
```
"我注意到一个模式：

所有成功的创作者都...

而失败的人都..."
```

---

## 🎭 情感共鸣型

### 挫折共鸣
- "感觉 ____ 吗？你不是一个人"
- "____ 真的很累"
- "如果你也在经历 ____"

### 激励鼓舞
- "如果你今天感到 ____"
- "给正在 ____ 的你"
- "____ 的你，值得被看见"

**示例：**
```
"如果你今天感到焦虑：

深呼吸。
你比昨天更强大了。

这里有一些方法..."
```

---

## 🎲 挑战/承诺型

### 挑战
- "我接受 X 天 ____ 挑战"
- "X 天 ____ 实验"
- "让我们一起 ____"

### 承诺
- "我保证 ____"
- "如果做不到 ____, 我就 ____"
- "X 天后给你结果"

**示例：**
```
"30 天内容创作挑战：

每天发 1 条 tweet
记录数据
30 天后分享结果

Day 1/30 👇"
```

---

## 🃏 幽默/反转型

### 预期违背
- "____ 很简单，只要你 ____"
- "如何 ____：第 1 步 ____, 第 2 步 放弃"

### 自嘲
- "我是 ____ 的专家（失败的那种）"
- "X 件 ____ 教我的事（大部分很痛）"

**示例：**
```
"如何创业成功：

第 1 步：有个 idea
第 2 步：辞职
第 3 步：????
第 4 步：profit

（开玩笑的，真实指南如下）"
```

---

## 📋 Hook 选择指南

### 根据内容类型选择

| 内容类型 | 推荐 Hook 类型 |
|---------|---------------|
| 教程/指南 | 直接利益型、数字型 |
| 个人故事 | 个人故事型、反常识型 |
| 热点评论 | 热点借势型、问题型 |
| 观点分享 | 反常识型、洞察型 |
| 激励内容 | 情感共鸣型、挑战型 |

### 根据目标受众选择

| 受众 | 推荐 Hook 类型 |
|-----|---------------|
| B2B 专业人士 | 数据型、洞察型 |
| 创业者 | 反常识型、个人故事型 |
| 学习者 | 问题型、直接利益型 |
| 普通用户 | 情感共鸣型、幽默型 |

## 🎯 Hook 强度评分

### 🔥🔥🔥🔥🔥 (必点)
- 强烈对比
- 反常识
- 个人冲突
- 悬念强

### 🔥🔥🔥🔥 (高概率点)
- 清晰利益
- 好奇心引发
- 情感共鸣

### 🔥🔥🔥 (可能点)
- 信息价值
- 温和好奇

### 🔥🔥 (难吸引)
- 平淡陈述
- 自嗨内容

### 🔥 (不会点)
- 广告感强
- 毫无吸引力

---

## 📝 Hook 组合技巧

### 双 Hook（开头 + 结尾）
- 开头抓眼球
- 结尾留悬念/CTA

### Hook + 价值承诺
- Hook 引发好奇
- 立即承诺价值（"这篇文章会告诉你..."）

### Hook + 社会证明
- Hook 引发好奇
- 立即给出证明（"X 万人看过"）

---

## ❌ 避免的 Hook

- 太笼统（"今天我想分享..."）
- 广告感强（"购买我们的产品..."）
- 自嗨（"我觉得..."）
- 太长（超过 2 行）
- 没有情感/利益/好奇

---

*好的 Hook 是 Viral 内容的一半*
