# Viral 内容模板库

基于 X 算法的 Viral 内容模板。

## 🧵 Thread 模板

### 模板 1：数字清单型
```
1/ [Hook：吸引人的开头]

我做了 [X] 个月 [某事]，总结出了 [N] 个 [技巧/教训]。

[具体承诺价值]

Thread 🧵

2/ 1️⃣ [要点1标题]

[详细解释]

[例子/数据]

3/ 2️⃣ [要点2标题]

[详细解释]

[例子/数据]

... (继续到 N)

[N+1]/ 总结：

[核心要点回顾]

[CTA：互动提问]

[标签]
```

**适用场景：**
- 教程/指南
- 经验总结
- 资源清单

**算法优势：**
- 结构化易读
- 数字引发好奇
- 每条都有价值

---

### 模板 2：故事叙述型
```
1/ [Hook：戏剧化开头]

[X 时间前]，我 [当时状况]。
[转折]
现在 [现在状况]。

这是发生了什么 👇

2/ 那时候...

[背景铺垫]
[当时的困境]

3/ 转折点发生在...

[关键事件]
[当时的心理活动]

4/ 我做了什么：

[行动1]
[行动2]
[行动3]

5/ 结果是：

[具体成果，最好有数据]
[对比 Before/After]

6/ 学到的教训：

[教训1]
[教训2]
[教训3]

7/ 如果你也在 [类似处境]，记住：

[鼓励的话]

[CTA]
```

**适用场景：**
- 个人成长
- 创业故事
- 转型经历

**算法优势：**
- 情感共鸣
- 完读率高
- 易引发讨论

---

### 模板 3：热点跟进型
```
1/ [热点事件] 发生了。

[X 分钟/小时] 看完要点：

[核心信息]

[为什么重要]

2/ 关键信息：

• [要点1]
• [要点2]
• [要点3]

3/ 这意味着什么？

[分析]
[影响]

4/ 你应该怎么做：

[行动建议1]
[行动建议2]

5/ 我的看法：

[独特观点]

[提问引发讨论]
```

**适用场景：**
- 新闻解读
- 产品发布
- 行业事件

**算法优势：**
- 时效性强
- 搜索流量
- 转发率高

### 模板 4：争议观点型
```
1/ [反常识观点]

[解释为什么这样说]

[引发好奇]

2/ 大多数人认为：

[主流观点]

3/ 但真相是：

[你的观点]
[证据1]
[证据2]

4/ [案例/数据支撑]

[具体例子]
[对比数据]

5/ 这意味着：

[结论]
[行动建议]

6/ 你同意吗？

同意扣 1，不同意扣 2

[继续讨论]
```

**适用场景：**
- 行业观点
- 方法论讨论
- 趋势判断

**算法优势：**
- 高讨论度
- 情感投入
- 易传播

## 📝 单条 Tweet 模板

### 模板 1：数据对比型
```
[大数字] vs [小数字]

[解释对比]

[核心洞察]

[CTA]

#标签
```

**示例：**
```
6 个月 vs 6 小时

我花了 6 个月学会编程，
上周用 AI，6 小时完成了同样的项目。

编程的门槛正在消失。

你想用 AI 做什么？

#AI #Coding
```

---

### 模板 2：问题反问型
```
[反问句]？

[解释问题]

[你的答案/观点]

[CTA]

```

**示例：**
```
为什么聪明人不分享他们的学习方法？

因为他们怕你学会了超过他们。

但我不一样。

这 5 个技巧让我效率提升了 10 倍 👇

#Productivity
```

## 🎣 Hook 模板

### 数字对比 Hook
- "X 年前我 ____，现在我 ____"
- "用了 X 天 ____，结果 ____"
- "X% 的人不知道 ____"
- "从 X 到 Y，我只用了 Z"
- "X 小时 vs X 个月"

### 反常识 Hook
- "不要再 ____ 了"
- "____ 是个谎言"
- "____ 正在毁掉你"
- "你一直被误导了"
- "停止做 ____"

### 个人故事 Hook
- "我被 ____ 了"
- "上周我 ____"
- "X 年前我不信 ____, 现在我 ____"
- "最糟糕的一天教会了我 ____"
- "没人相信我能 ____"

### 问题悬念 Hook
- "为什么 ____?"
- "____ 的秘密是什么?"
- "如果 ____ 怎么办?"
- "你知道 ____ 吗?"
- "____ 的真正原因是?"

### 直接利益 Hook
- "如何 ____"
- "X 个方法让你 ____"
- "停止 ____, 开始 ____"
- "用 X 分钟学会 ____"
- "____ 的终极指南"

## 💬 互动设计模板

### 直接提问
- "你怎么看？"
- "同意吗？"
- "你的经验是什么？"
- "遇到过这种情况吗？"

### 选择题
- "A 还是 B？"
- "1 还是 2？"
- "同意扣 1，不同意扣 2"

### 投票式
- "点赞过 X 发下一条"
- "评论区超过 X 条分享细节"

### 求助式
- "有人知道 ____ 吗？"
- "求推荐 ____"
- "该怎么选？"

## 📊 内容评分标准

### 9-10分（必火）
- Hook 抓眼球
- 价值密度高
- 有情感共鸣
- 有互动设计
- 时间完美

### 7-8分（高概率火）
- Hook 不错
- 有价值
- 互动设计一般

### 5-6分（可能火）
- 需要优化 Hook
- 价值不够突出

### 3-4分（难火）
- 平淡无奇
- 需要重写

### 1-2分（不会火）
- 自嗨内容
- 无价值传递
