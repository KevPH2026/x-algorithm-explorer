> 理解算法 → 反向工程 → 生成算法喜欢的内容

## 🚀 核心功能

### Viral Post 生成器
基于 X 算法特征，生成高传播概率的内容：
- ✅ 算法友好的内容结构
- ✅ 高互动率的开头设计
- ✅ Thread 最佳实践
- ✅ 标签和发布时间建议

### Hook 生成器
前 3 秒抓眼球的开头：
- 数字对比型
- 反常识型
- 个人故事型
- 问题悬念型

### 内容优化器
分析并优化已有内容：
- 流量潜力评分
- 改进建议
- A/B 测试版本

### 热点追踪
识别当前算法偏好的话题类型

## 📊 X 算法核心洞察

### 算法偏好的内容特征

| 特征 | 说明 | 应用 |

| **时间衰减** | 新内容优先 | 实时热点蹭流量 |
| **个性化** | 基于用户兴趣图谱 | 标签精准匹配 |
| **权威性** | Tweepcred 影响分发 | 专业背书、数据支撑 |

```
Viral Score = Hook (30%) + Value (40%) + Engagement (20%) + Timing (10%)
```

## 🎨 Hook 模板库

### 数字对比型
- "X 年前我 ____，现在我 ____"
- "用了 X 天 ____，结果 ____"
- "X% 的人不知道 ____"

### 反常识型
- "不要再 ____ 了"
- "____ 是个谎言"
- "____ 正在毁掉你"

### 个人故事型
- "我被 ____ 了"
- "上周我 ____"
- "X 年前我不信 ____, 现在我 ____"

### 问题悬念型
- "为什么 ____?"
- "____ 的秘密是什么?"
- "如果 ____ 怎么办?"

## 📅 最佳发布时间

| 类型 | 最佳时间 | 原因 |
|------|---------|------|
| B2B 内容 | 周二-周四 9-11 AM EST | 工作时间，专业用户活跃 |
| 娱乐内容 | 晚上 8-10 PM EST | 下班放松时间 |
| 热点跟进 | 发布后 2-6 小时内 | 流量窗口期 |
| Thread | 上午发布 | 给用户时间读完 |
